## 📖 Что такое белые и чёрные списки?

Большинство людей привыкли к **чёрным спискам**: провайдер блокирует отдельные сайты или сервисы по требованию властей.  
Но в последнее время на смену приходят **белые списки** — принципиально иной уровень контроля.

| Критерий | ⚫ Чёрный список (Blacklist) | ⚪ Белый список (Whitelist) |
|----------|-----------------------------|----------------------------|
| **Как работает** | Блокируются конкретные адреса (IP/домены), всё остальное доступно | Разрешены только сайты из списка; всё остальное недоступно по умолчанию |
| **Объём блокировки** | Тысячи или десятки тысяч ресурсов | Миллионы сайтов закрыты, открыты единицы |
| **Сложность обхода** | Достаточно обычного VPN/прокси | Нужны спецпротоколы, которые маскируются под разрешённый трафик |
| **Пример** | Запрет отдельного новостного сайта | Доступ только к госуслугам, почте и паре поисковиков |
| **Где применяется** | Большинство стран с интернет-цензурой | Некоторые провайдеры в РФ в тестовом режиме, корпоративные сети |

**Когда провайдер вводит белый список, ваш обычный VPN перестаёт работать** — его трафик просто отбрасывается, потому что он не входит в список «разрешённых».  
Наши конфиги созданы специально для таких ситуаций: они обходят DPI, имитируют безобидный HTTPS‑трафик и остаются незаметными для фильтров.
